# Minecraft.Helper.website
# ⛏️ Minecraft Helper

**Just a site for Minecraft players**

## 📖 О проекте
Этот сайт создан для русскоязычных игроков Minecraft. Здесь вы найдете:
- Информацию об игре
- Полезные советы для новичков
- Систему регистрации
- (в будущем) Крафт верстак, зельеварку и консольные команды

## 🛠️ Технологии
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 🎯 Цель проекта
Помочь игрокам Minecraft быстро находить нужную информацию и учиться новым механикам игры.
